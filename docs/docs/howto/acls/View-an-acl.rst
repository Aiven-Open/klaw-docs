View Acls
====================
All acls can be viewed underneatht the topic they are associated with. Navigate to the **Topics** menu in Klaw and select the topic for which you want to view the acls.
When viewing the ACls you will notice that acls not owned by your team have IP and Principal/Username details masked.
When viewing an acl that your team owns you will be able to see this IP and Principal/Username information but also you will ave the ability to retrieve the password and also create a request to delete your acl if desired.
